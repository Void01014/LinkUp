# LinkUp

LinkUp is a user-friendly, modern social platform where people can easily connect, share their lives, and communicate in real-time. Whether you are posting your latest updates, forming close-knit friend circles, or chatting directly with peers, LinkUp provides a seamless and engaging experience.

## 🛠️ Tech Stack

This project is built using a robust and modern technology stack:

- **Backend:** PHP 8.2+, Laravel 12
- **Frontend:** Livewire, Alpine.js, Tailwind CSS
- **Real-time Communication:** Laravel Reverb, Laravel Echo, Pusher JS
- **Authentication & Tooling:** Laravel Breeze, Laravel Socialite
- **Assets & UI:** Vite, SweetAlert2

Securely register and log in using your existing Google or GitHub accounts.

### 🟢 Live Status & Read Receipts
Stay connected with a live status indicator bubble showing who is online or offline. Real-time read receipts let you know exactly when your messages have been seen.

### ⏱️ Disappearing Messages
For enhanced privacy, enable disappearing messages to automatically delete chat history after 24 hours.

### 🤝 Friend Circles
Form exclusive friend circles by easily sending, accepting, or declining invites. You can also add friends quickly by scanning their unique QR code or sharing an invite link.
